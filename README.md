# To-do List

Uma lista de tarefas simples construída com HTML, CSS e JavaScript.

Acesse o projeto: <a href="https://milennafeijo.github.io/to-do-list/">aqui</a>.

## Funcionalidades

- Adicionar novas tarefas.
- Marcar tarefas como concluídas com checkbox.
- Excluir tarefas individualmente.
- Salvar tarefas no `localStorage` do navegador para manter a lista mesmo após recarregar a página.
- Atalho `Enter` para adicionar tarefas rapidamente.

## Como usar

1. Abra o arquivo `index.html` no seu navegador.
2. Digite uma tarefa no campo de entrada.
3. Clique em **Adicionar** ou pressione `Enter`.
4. Use o checkbox para marcar a tarefa como concluída.
5. Clique em **Excluir** para remover a tarefa.

## Arquivos principais

- `index.html` — estrutura da página.
- `styles.css` — estilos visuais.
- `script.js` — lógica de adicionar, remover e salvar tarefas.
